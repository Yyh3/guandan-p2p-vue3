## 二、v1.0 基础可用(2026-06-08 完工)

**目标**:跑通核心游戏循环,4 标签可联机。

**完成项**:
- 牌组生成 / 洗牌 / 发牌(108 张 / 4 人 × 27 张)
- 牌型识别(14 种:单/对/三/三带二/顺/连对/钢板/4-8 炸/同花顺/王炸)
- 大小比较(`canBeat`)
- 逢人配(红桃级牌做万能)
- AI 出牌(规则 + 贪心)
- 对局状态机(发牌 / 出牌 / 过牌 / 一轮结束 / 一局结束)
- 升级 + 进贡计算
- 浏览器版网络层(BroadcastChannel)
- localStorage 存储(昵称/头像/设置/战绩)
- Vue 3 + Vite 工程化
- 80+ 单元测试

**已知限制**(v1.0 范围内不修):
- 网络层只支持同浏览器标签联机
- AI 只有中等难度
- 没原生 APK/IPA 打包
- 无录像回放

---

## 三、v2.0 真机联机(设计中)

**目标**:4 手机移动热点对局,真机原生打包。

**核心工作**:
- TCP Socket 网络层(替换 BroadcastChannel)
- Capacitor 集成(Android + iOS)
- 原生 TCP 插件(Android:Java/Kotlin;iOS:Swift Multipeer)
- 真机键盘 / 触屏适配
- 移动端 UI 重构(从桌面 1280×800 → 移动 375+)

**风险**:
- Capacitor 平台分支管理
- iOS Multipeer Connectivity 配置复杂
- 移动端 UI 改动大,可能影响桌面端

**预计**:2026 Q3-Q4(看资源)

---

## 四、v3.x UI/UX 优化(2026-06 已完成)

**目标**:把 v1.0 的「能玩」打磨成「想玩」。

**v3.0-3.4**:核心 UI 重设计
- 顶部 HUD 重组(房间号/级牌/倒计时/座位)
- 中央牌桌 / 桌面牌 / 首家提示
- 手牌竖叠排版(每列一个 rank)
- 身份三色(自己绿/队友蓝/对手红)
- 设计 token 系统(tokens.css)

**v3.5**:Bug 修复 + 体验小改
- 智能理牌按钮
- 一键理(调 AI)
- 提示模式

**v3.6**:更多视觉打磨
- 4 角单字(左上玩家名/右上倍数/左下级牌/右下轮数)
- 智能理牌渐变按钮
- 列底 ×N 数量提示
- 大小王 SVG 化(headless 渲染兼容)

**v3.7**(2026-06-08 完工):**体验优化三个等级**

**P0**(必做,体验门槛):
- 报数提示(剩 ≤ 10 张时显示张数,触发 tick / warn / urgent 音效)
- 对局中禁止改名(弹 toast)
- 炸弹 / 王炸 / 超炸音效(Web Audio 合成)
- BGM 折中方案(Web Audio 拍鼓点 280ms kick+hat+snare+贝斯)
- 自座位位置 C(底部居中突出,避开手牌区)

**P1**(增值,操作效率):
- 出牌音效分级(sfxSingle / sfxPair / sfxStraight 按节奏 + 音高阶梯)
- 倒计时 ≤ 5s 蜂鸣 + 手牌区闪红
- 桌面端快捷键(A=提示 / Space=出牌 / P=不出 / Esc=菜单,input 焦点时让权)
- 快捷聊天 10 颗 pill(本地 toast,v2.0 留 P2P 接口)

**P2**(完善,管理):
- 战绩图表(零依赖 SVG 柱+折线)
- 集中设置面板(BGM/SFX 开关 / 音量 / 风格 / 清空战绩)
- `/settings` 路由 + 首页"设置"入口

**Bug 修复**:
- 动画开关 disabled 状态不再显示绿色(改灰)
- BGM 风格持久化到 localStorage

## 五、v3.8 计划(待启动)

**目标**:补 v3.7 留下的尾巴 + 用户反馈

**候选**:
- 快捷聊天 P2P 同步(v3.7 留了 v2.0 接口注释)

2. 聊天 P2P(打通 v3.7 留的接口)
3. 移动端适配(v2.0 前置)

## 六、v4.0 AI 难度(构思中)

**目标**:Easy / Medium / Hard 三档 AI

**实现方向**:
- **Easy**:当前 `decide()` + 30% 概率随机 pass
- **Medium**:当前 `decide()`(规则 + 贪心)
- **Hard**:1 层博弈搜索(枚举对手可能的反压,选期望最优)+ 剩余牌推断

**预计**:v3.8 后启动

## 七、长期目标

- **4 人局域网掼蛋的"事实标准"** — 离线场景首选
- **多端覆盖** — H5 + Android APK + iOS IPA + Windows / macOS(electron 可选)
- **规则覆盖** — 经典掼蛋 / 升级掼蛋 / 自由组局(预留扩展点)
- **赛事化** — 录像回放 + AI 复盘 + 计分系统(可选)

---

## 八、非目标(明确不做)

> 这些是**故意**不做的,别在新功能里加。

- ❌ **联网对局** — 破坏离线原则
- ❌ **账号系统 / 登录** — 破坏纯净绿色
- ❌ **充值 / 内购 / 广告** — 同上
- ❌ **聊天系统** — 只做快捷短语(本地 toast),v2.0 接 P2P
- ❌ **社交分享** — 破坏纯净
- ❌ **排行榜** — 破坏单机
- ❌ **用户注册 / 实名** — 破坏纯净
- ❌ **依赖第三方 UI 库** — 保持极简 / 版权干净
- ❌ **AI 深度学习** — 改 AI 方向用户拍板了
