# v3 UI 重设计 Spec（基于用户提供的 3 张商业掼蛋截图）

> 实施日期：2026-06-06
> 设计参考：用户提供的 3 张商业掼蛋游戏截图（见 `.harness/designs/screenshots/`）
> 目标：把当前 GameView 重做成商业级视觉效果

---

## 一、设计目标

参照商业掼蛋游戏的视觉风格：
- **俯视圆桌 + 蓝色椭圆牌桌 + 木边**（已经有，要加强）
- **橙黄强调色**（按钮 / HUD 卡片 / 倒计时）
- **4 玩家座位头部信息**（头像 / 名字 / 金币 / 等级 / 标签）
- **顶部 HUD 多卡片**（本局打 / 倍数 / 倒计时 / 记牌器）
- **中央特效**（炸弹 / 王炸 / 不出飘字）
- **主操作栏浮起**（不出 / 提示 / 出牌）
- **手牌更精致**（JOKER 大字 + 级牌红"级"字 + 红/黑分明）

## 二、关键元素清单

### 1. 顶部 HUD 区（3 部分）

#### 1.1 左上：本局打 + 倍数 卡片
- 橙黄渐变背景 + 圆角 8px
- 数字 36-44px 粗体
- 标签 12px 半透明白
- 间距 8px
```
┌─────────┐  ┌─────┐
│ 本局打  │  │ 倍数 │
│   10    │  │  2  │
└─────────┘  └─────┘
```

#### 1.2 顶部居中：4 玩家头像座位
- 4 个头像方框分布：上 / 下 / 左 / 右
- 每个头像：方框 50×50 + 圆角 8px + 文字信息
  - 头像 emoji 或图片
  - 名字（最多 6 字 + ...）
  - 金币数（💰 + 数字 + 万/亿单位）
  - 等级（LV + 数字 + 名称）
  - **队友**：黄底 + "队友" 角标
  - **自己**：绿底 + 金币 + 加号（点击加好友）
  - **对手**：浅灰底
- 头像可点击（弹出菜单：踢人 / 私聊 / 拉黑）

#### 1.3 顶部右上：4 个图标按钮
- 对局（数字 0/2 徽章）
- 牌型
- ···（更多）
- 设置（齿轮）
- 圆形 36×36 + 灰底

#### 1.4 倒计时闹钟（顶部居中偏右）
- 橙色卡片 48×48
- 闹钟 emoji + 数字
- 数字 ≤10s 时变红色 + 抖动
- 数字 0 时显示闹钟 + "已超时"

#### 1.5 记牌器（顶部右侧）
- 4 行 × 13 列 网格
- 行：♠♥♣♦（4 花色）
- 列：A K Q J 10 9 8 7 6 5 4 3 2（13 rank，从大到小）
- 每格：12px 数字 = 剩余张数（0-4）
- 0 = 已全部出过（灰）
- 1-4 = 剩余（彩色：红桃/方块红，梅花/黑桃黑）
- 鼠标悬停：显示"已出 X 张"tooltip
- 右侧"同花顺"提示框（默认折叠，鼠标悬停时展开）

### 2. 中央牌桌区

#### 2.1 椭圆牌桌
- 背景：径向渐变（中心深蓝 #1a2a5e → 边缘深紫 #2a1a4e）
- 椭圆 70% 宽 × 60% 高（屏宽自适应）
- 木边：边框 4px 棕色 #8B5A2B + 内阴影模拟立体感
- 桌面上可加丝印"爱掼蛋"水印（半透明）

#### 2.2 桌面牌堆
- 多张牌错位堆叠：每张偏移 14px（水平）+ 3° 随机旋转
- 已出牌最多显示最近 1 轮（6-8 张），超过 1 轮的淡出
- 点击桌面牌可放大查看

#### 2.3 特效层（覆盖在桌面上）
- **超级炸弹 / 炸弹 / 王炸 / 顺子 / 同花顺**：
  - 屏幕中央大字 1.5s（火焰渐变 + 阴影 + 缩放弹跳）
  - 屏幕震动 0.4s × 2
  - 不同牌型不同特效
- **"不出"飘字**：
  - 对应座位飘出气泡（黄底 + "不出"）
  - 1.2s 淡出 + 向上飘
- **"过牌"飘字**：类似"不出"但显示"过牌"

### 3. 玩家手牌区（底部）

#### 3.1 手牌布局
- 27 张错位：每张牌宽 50px + 间距 -30px（重叠）
- 玩家可点击选中（高亮 1-2px 金色边框 + 抬升 8px）
- 选中的牌顶部出牌（从手牌区向上"飞"到桌面）

#### 3.2 牌面设计（参考截图）
- 牌大小 60×84px
- 白底 + 圆角 6px + 1px 灰边 + 浅阴影
- 4 角设计：
  - 左上：小数字（10px）+ 小花色（12px）
  - 右下：大数字（24px 粗体）+ 大花色（28px）
  - 中下方：大花色（40px 半透明）
- 颜色：
  - 红桃 / 方块：红色 #E74C3C
  - 梅花 / 黑桃：黑色 #2C3E50
- **级牌特殊**：
  - 4 角小数字 / 大数字 / 花色照常
  - **中下方大花色** 改大花色 + 红色"级"字
  - **背景** 改黄色 #FFD54F 半透明
  - **边框** 改金色 #FFA000
- **JOKER**：
  - 大字 JOKER 24px 红色 + Joker 角色（猴王图案）
  - 边框金色 + 圆角 8px

#### 3.3 手牌上方：底部花色 tab
- ♣♠♥♦ 4 个按钮
- 圆 32×32 + 灰底
- 点击：手牌中该花色的牌自动选中

#### 3.4 主操作栏（出牌时浮起）

  - **不出**（黄 #FFC107）— 240×56
  - **提示**（蓝渐变）— 240×56

#### 3.5 底部右侧：3 个圆形按钮

## 四、动效规范

- **出牌飞行动画**：手牌 → 桌面，300ms cubic-bezier(0.4, 0, 0.2, 1)
- **发牌 stagger**：每张 80ms，总 ~2s
- **炸弹特效**：屏幕中央大字 1.5s（缩放 0.5→1.1→1）+ 震动 0.4s
- **不出飘字**：从对应座位向上飘 30px + 淡出 1.2s
- **倒计时抖动**：≤10s 时 transform: translateX(±2px) 0.1s 无限循环
- **记牌器数字变化**：scale 1.2 → 1.0 200ms
- **按钮 hover**：filter: brightness(1.1)
- **按钮 active**：scale 0.95

## 五、风格约束（必须遵守）

- **零外部依赖**：不引入图标库 / 字体库 / 动画库——全部手写或内联
- **保持 ESM**：import/export，不引入 require
- **Vue 3 `<script setup>` 组合式 API**
- **CSS 用 scoped**
- **所有公共函数 1 行注释 + 参数/返回值**
- **保持现有功能**：发牌动画 / BGM / 出牌声效 / 智能理牌 / 一键理 全部不能被破坏
